# scheme プログラム
Mac OS X version 10.10.3 で動かしてます。

## 使い方
プログラムの置いてあるディレクトリに移動してください。
scheme 起動後
```scheme
(load "file_name")
```
です。
それぞれのプログラムの使い方は、load 時に表示されます。

## 一覧

### plot_2d_graph.scm
1変数関数を2次元上にプロットする

### plot_2d_graph_recursive.scm
1変数関数を2次元上にプロットする(上の再帰的プロセスを用いた版)

### plot_3d_graph.scm
2変数関数を3次元上にプロットする
